Sobre mim

Sou uma pessoa comprometida, organizada, atenciosa, com sede de conhecimento, e capaz de lidar com situações desafiadoras de forma eficiente, tento demonstrar meu empenho em aprender atravéz dos meus projetos, procurando me ajustar a novas tarefas de forma criativa.

Nos últimos 2 anos de faculdade, onde mergulhei no mundo da tecnologia, saindo do meu antigo emprego para me dedicar totalmente a isso, comecei obtendo conhecimento com lógica de programação, e também adquirindo habilidades em desenvolvimento web, com conhecimentos sólidos em frontend, HTML, CSS, Javascript, React, utilizando API's Rests para projetos e banco de dados utilizando o MySQL.

Também, me aventurei nas linguagens Backend, como NodeJS, Java e Python, e atualmente, busco me atualizar neste campo. Meu objetivo, é entrar no mercado de trabalho e utilizar meu conhecimento para auxiliar empresas e usuários nas suas aplicações.

Calculadora Javascript

Este projeto é uma calculadora interativa desenvolvida utilizando HTML, CSS e JavaScript. Ele tem como objetivo proporcionar uma experiência prática e intuitiva para cálculos matemáticos básicos, como adição, subtração, multiplicação e divisão.

Este projeto é um aplicativo de previsão do tempo desenvolvido com HTML, CSS e JavaScript. Ele utiliza dados de uma API de clima em tempo real para fornecer informações atualizadas sobre as condições meteorológicas de qualquer cidade.

Conversor de moedas

O Conversor de Moedas é um aplicativo web desenvolvido com React que permite aos usuários converter valores entre diferentes moedas de forma simples e intuitiva. O aplicativo utiliza uma interface interativa e dinâmica.
